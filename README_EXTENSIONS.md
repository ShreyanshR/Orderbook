# C++ IntelliSense Setup

## Required Extensions

To get code suggestions, syntax highlighting, and IntelliSense for C++, install these extensions in VS Code/Cursor:

### 1. **C/C++ Extension** (Essential)
- **Extension ID**: `ms-vscode.cpptools`
- **Name**: C/C++ by Microsoft
- Provides: IntelliSense, debugging, code browsing

### 2. **CMake Tools** (Recommended)
- **Extension ID**: `ms-vscode.cmake-tools`
- **Name**: CMake Tools
- Provides: CMake integration, build support

### 3. **CMake** (Syntax highlighting)
- **Extension ID**: `twxs.cmake`
- **Name**: CMake
- Provides: CMake syntax highlighting

## Installation

### Option 1: Via Extension Marketplace
1. Open VS Code/Cursor
2. Press `Cmd+Shift+X` (Mac) or `Ctrl+Shift+X` (Windows/Linux)
3. Search for "C/C++" and install
4. Search for "CMake Tools" and install

### Option 2: Via Command Line
```bash
code --install-extension ms-vscode.cpptools
code --install-extension ms-vscode.cmake-tools
code --install-extension twxs.cmake
```

### Option 3: Auto-install (Recommended)
The `.vscode/extensions.json` file will prompt you to install recommended extensions when you open the workspace.

## Configuration

The project is already configured with:
- `.vscode/settings.json` - C++ IntelliSense settings
- `compile_commands.json` - Generated by CMake for accurate IntelliSense

## After Installation

1. **Reload the window**: `Cmd+Shift+P` → "Reload Window"
2. **Select a C++ file** - IntelliSense should start working
3. **Wait for indexing** - The C++ extension will index your codebase (check status bar)

## Verify It's Working

1. Open any `.cpp` or `.hpp` file
2. Type `std::` - you should see autocomplete suggestions
3. Hover over symbols - you should see type information
4. Red squiggles should appear for errors

## Troubleshooting

### No IntelliSense?
1. Check if `compile_commands.json` exists in the project root
2. If not, run: `cd build && cmake -DCMAKE_EXPORT_COMPILE_COMMANDS=ON ..`
3. Reload the window

### Wrong IntelliSense?
1. Press `Cmd+Shift+P` → "C/C++: Reset IntelliSense Database"
2. Reload the window

### Still not working?
1. Check the Output panel: View → Output → Select "C/C++"
2. Look for error messages
3. Make sure you have a C++ compiler installed (`clang++` or `g++`)
